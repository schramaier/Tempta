Conclusão

A conclusão é uma das partes consideradas mais fáceis da redação. Nela, você irá fazer uma proposta de intervenção que resolverá um dos teus problemas que você argumentou nos parágrafos de desenvolvimento. Essa proposta deve respeitar os direitos humanos e ter os seguintes elementos:

  • Quem/ Agente : Quem irá resolver esse problema?
  • O que/ Ação : O que será feito para resolver esse problema?
  • Como/ Modo ou meio : Como será feito essa solução?
  • Por que/ Efeito : Qual o efeito de realizar essa ação?
  • Pra que/ Detalhamento : Detalhe algum dos itens acima.
Além de uma leve retomada do problema que você irá achar uma solução para resolver.
